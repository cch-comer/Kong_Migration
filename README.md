Kong_gui based on https://github.com/Kong/kong-manager.git

### frontend: Vue.js
### backend: Spring boot

# needs
1. node Version : v20.18.0
2. JDK 18 1

# How to Use

## Run once when cloned
1. cd src/front
2. npm install -g pnpm
3. pnpm install


## Run Server

1. pnpm vite

## Run Build
1. pnpm vite build
2. copy all files src/front/dist/* to src/main/resources/static/*
